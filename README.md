# Simple Python Calculator
This repo is a simple Python-based calculator that can perform basic arithmetic operations such as addition, subtraction, multiplication, division, exponentiation, and floor division.

## Description
The calculator takes two numbers and an operation as input, performs the arithmetic operation.
It supports common math operators like '+', '-', '*', '/'

## Features 
- Addition
- Subtraction
- Multiplication
- Division

## How to Use
1. Clone this repository.  
2. Open the Python file in your code editor (IDE).  
3. Run the program in your IDE.  
4. Enter two numbers and an operator to get the result.

## Project Info
- **Language:** Python  
- **Level:** Beginner  
- **Purpose:** Learn basic arithmetic functions using user-defined functions.
